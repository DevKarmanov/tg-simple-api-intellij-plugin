Detects usage of unregistered actionName values in addActionData(Set.of(...)) calls.

This inspection highlights action names that are not registered in any known handler method annotated with supported Telegram annotations, such as @BotCallBack, @BotText, etc.

Use this inspection to avoid silent errors caused by typos or missing handlers, which could lead to incomplete or broken conversation flows in Telegram bots. 


// Example: this will be highlighted if "foo-action" is not registered
